# Yılan Oyunu

C ve SDL3 kütüphanesi ile geliştirilmiş klasik yılan oyunu.

## Özellikler

- Yön tuşlarıyla yılan kontrolü
- Yem toplama ve uzama
- Skor gösterimi
- Game over ve yeniden başlama ekranı

## Gereksinimler

- SDL3
- SDL3_ttf

Arch Linux'ta kurulum:
```bash
sudo pacman -S sdl3 sdl3-ttf
```

## Derleme

```bash
make
```

## Oynanış

- Yön tuşları: yılanı yönlendir
- Q: çık
- R: yeniden başlat (game over ekranında)
